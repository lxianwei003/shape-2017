Powered by IBM Watson

Using Watson's Natural Language Understanding Api, we are able to extract keywords from the websites that we have visited in the past. We then match these keywords with the keywords saved in our data base and then find a movies with the best possible match

We used The Movie Database.

The Movie Database allowed us enabled us to have a better selection of movies using certain keywords to provide a list of recommended movies.

Our Goal

We want to provide a convenient means of finding entertainment. Instead of having to click and scroll through multiple websites to find a good movie, Movie pup suggests movies with a single click.

Future of this project

We are planning to perfect the selection algorithm. Being able to increase the efficiency of the algorithm allows it to provide a more accurate list of recommendations.
